# Quoro Generator

## Description
The Quoro Generator is a simple web application that generates random quotes and their respective authors. It's a fun and inspirational tool to discover new quotes and authors.

## Features
- Randomly generates quotes and authors.
- Click the "New Quote" button to get a new random quote.
- Inspirational quotes to motivate and inspire you.
- Easy-to-use and user-friendly interface.
